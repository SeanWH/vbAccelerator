vbAccelerator COM/VB Support Type Library (vbaCOM.Tlb)

This Type Library, part of the larger vbAccelerator COM Support Library project, provides access to many of the underlying ActiveX control run-time and design-time interfaces.

The following interfaces are exposed:

  • IDispatch
  • IEnumOLEVERB
  • IEnumVARIANT
  • IErrorLog
  • IOleClientSite
  • IOleControl
  • IOleControlSite
  • IOleInPlaceActiveObject
  • IOleInPlaceFrame
  • IOleInPlaceSite
  • IOleInPlaceUIWindow
  • IOleWindow
  • IPerPropertyBrowsing
  • IPerist
  • IPeristPropertyBag
  • IPeristStreamInit
  • IPropertyBag
  • IPropertyPage
  • IPropertyPageSite
  • ISequentialStream
  • IStream
  • ITypeInfo
  • ITypeLib

To see the Type Library in use, try the following articles:

  • For the IOleControl interface, the Button List Bar Control and the Using IOleControl article.
  • For the design time interfaces like IPerPropertyBrowsing, see the main vbAccelerator UserControl COM Support Library article.
