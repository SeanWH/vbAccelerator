GDI+ Type Library (GDIPlus.TLB)

This Type Library provides all the declares, enumerations and structures used for working with GDI+ in VB.

This Type Library includes all of the GDI+ declares from the Platform SDK GdiPlusFlat.h file, plus structures and enumerations from GdiPlusImaging.h, GdiPlusHeaders.h as well as general Win API declares and types to support using the library.

In order to use GDI+ you must have the GDI plus package installed on your machine. This is installed by default on Windows XP machines, and available as a freely redistributable package for NT4 sp6, Windows 2000, 98 and ME from Microsoft here:

GDI Plus Redistributable

To see the library in use, check out:

  • Dana Seaman's postings at Planet Source Code (http://www.Planet-Source-Code.com/vb/scripts/ShowCode.asp?txtCodeId=42861&lngWId=1)
