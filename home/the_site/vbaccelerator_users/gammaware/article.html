GammaWare LVS NG

LVS NG is fast and modular Warehouse Management System. It offers a particularly easy to use interface with a lot of search power and has been developed in partnership with Gammaware's existing customer base to improve functionality and usability.

Using vbAccelerator Code

LVS NG uses the following vbAccelerator controls to enhance the user interface:

  • S-Grid Control
  • ListView Control
  • ImageList Control
  • ExplorerBar Control

About GammaWare

More information about GammaWare and LVS NG can be found at their website (German language):
