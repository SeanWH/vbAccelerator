vbAccelerator Users Index

vbAccelerator controls and code are being used in various applications around the world (not just my own!) This section provides a look at just a few applications out there using the code.

If you'd like to see your application or company mentioned here, please send me a mail with some information about which controls or code you're using and I'd be delighted to feature it.

IconArt

IconArt is a highly-compact cursor and icon creator. IconArt allows you to either draw the image from scratch using a variety of common drawing tools or to import an image to turn into an icon or cursor. IconArt has recorded well over 100,000 downloads since launch and has been published with various computer magazines.

GammaWare LVS NG

LVS NG is fast and modular Warehouse Management System. It offers a particularly easy to use interface with a lot of search power and has been developed in partnership with Gammaware's existing customer base to improve functionality and usability.

Brixoft SourceEdit - an extensible, multi-language programmers editor

Source Edit is an editor for developers which can of course be used as a regular text editor. It provides native supports for a whole range of programming languages, including ASP, PHP, Java, C/C++, C#, Visual Basic, Pascal, Perl, Cold Fusion, SQL, HTML, CSS and XML, and can be customised to support new languages using a Language Editor.

Source Edit's facilities include code colouring, Intellisense-style code completion, language-specific context sensitive help, a full hex editor, reading and writing directly to an FTP server and many other powerful features.

ZR Software's EzyTrans

EzyTrans is an tool that provides a enhancements for users of Quicken and Microsoft Money accounting software and even Excel. It is designed to vastly reduce the time taken to enter bank statements into accounting software by providing automatic import and management facilities for electronic bank statements. By both providing automatically assigned categories for transactions, and the ability to define customised search and replace criteria is is both comprehensively customisable and easy to use.

Last Updated: 1 November 2003

Office Navigator

Office Navigator is a an Office tool that remembers all the Office files you've opened in the past, and the last 20 folders where you've browsed. It lets you create point-and-click lists of all your favorite Office files, templates, and folders, so you never have to look for them again. In addition it turns the Task Pane into a powerful document and template launcher, and adds up to 255 icons to the Places Bar in the 'Open' and 'Save As' dialog box. By using a compact menu structure, it takes up only one button's screen real estate but yet provides immediate access to a vast array of documents - and works from any Office application.
