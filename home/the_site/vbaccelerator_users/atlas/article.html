Atlas

Great Looking software for PBX billing and control

Atlas is a Ukranian company offering PBX control and billing solutions as well as other corporate software projects. They are using customised vbAccelerator controls and code to provide very high quality user interfaces for their applications.

Using vbAccelerator Code

The applications use a variety of controls and code, including:

  • Toolbar/Rebar
  • PopupMenu DLL
  • Owner-Draw Combo and ListBox
  • TabStrip Control
  • XML Property Bag
  • StatusBar and Progressbar

PBX Control Application

One of the things Atlas find most useful about the controls is that full source code is always provided. This has allowed them to add new features as well as make changes in particular to ensure correct operation in a Russian language system.

Corporator Communicator IM prototype using NeoCaption for skinning
