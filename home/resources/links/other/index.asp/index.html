LAME - Lame Ain't an MP3 Encoder

LAME is a the source code for a fully GPL licenced MP3 encoder, which has both the speed and quality to rival commercial MP3 encoders. Since the project's inception in 1998 the code has been regularly improved and the feature list now includes MPEG 1, 2 and 2.5 layer III encoding, constant (CBR) and variable (VBR and ABR) bit-rate encoding and psycho-acoustic noise-shaping. Note that the project is distributed as C source only, however, you can obtain a binary version of the encoder by downloading CDEx.

CDEx

CDEx is a very nice tool for CD ripping which has been developped continually since 1998. Tracklisting is provided through FreeDb.org and audio files can be encoded in a wide variety of formats including MP3, Ogg Vorbis, Monkey's Audio as well as to standard WAV files. There are also some nice batch encode/decode screens for translating between audio formats.
