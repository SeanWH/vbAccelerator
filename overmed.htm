This section is devoted to Graphics and Multimedia programming for VB.

 
 
  • Saving VB Pictures to JPEG using the cDIBSection class and the Intel® JPEG Library. 8 February 2000
  • 256 Colour DIB Sections and Palette Manipulation If you want bitmap effects to run fast and use 256 colour images then this is the code you need. 5 October 1999
  • Image Processing and Real time Effects using DIB Sections - Demonstrates a greatly expanded Image Processing repetoire and some cool real time fade effects. 26 February 1999
    • Using DIB Sections and the cDIBSection Class Wrapper
    • Rank Filters
    • Hue Lightness and Saturation (HLS) Model and Manipulating Colours
    • Floyd-Stucci Colour Reduction Method and Gray Scaling
    • Texturising Images - a sample use of cDibSection
    • Colour Depth Control - saving a true Win32 BMP file with your chosen colour depth. Includes Octree Colour Quantisation code from Brian Schimpf.
  • Streaming .WAV file player - play wave files of any size asynchronously using the digital audio multimedia API.
  • High Resolution Multimedia Timer - a safe, VB implementation of the Multimedia timer - guaranteed timer event accuracy down to 1ms.
  • Image Processing in VB using the VB Picture object - Demonstrates a simple class for blurring, sharpening and embossing images.
  • Transparent Sprite Library - A full sprite implementation showing double buffering and providing a slightly mad screen saver which you can easily customise!

Projects under development:

  • MIDI in VB using the High Performance Timer.
  • DirectX Audio mixing, digital effects and streaming. Until then, please check out the fantastic stuff at Patrice Scribe's Area 51 site
  • Alpha blending bitmaps using the new Win98 bitmap functions (also NT5, which will probably have been released once I've finished this!) - I haven't even got Win98 yet! Want to help? The documentation is all there in MSDN under Platform SDK-> Graphics and Multimedia Services-> GDI-> Bitmaps. Looks cool!
