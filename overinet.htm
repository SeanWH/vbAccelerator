A small but gradually growing library of Internet related source code. In fact its growing much more gradually than I originally thought...

 
 

  • XML Property Bag A great code sample from Aaron Anderson allowing you to simply implement XML storage for any application.
  • Asynchronous Transfer Component - a simple but great idea! Manage as many asynchronous downloads from the Net as you want using this ActiveX DLL from Dion Wiggins.
  • The e-Checker application (30kb) from Julio Moreyra (jmoreyra@juschubut.gov.ar) checks for new mail on a POP3 server at pre-defined intervals and can open your preferred mail program.

Coming up!

  • A class to implement FTP file transfers using WININET.DLL calls. The code for this is now ready (and in production) but I need to do a bit more testing (hope the production app doesn't die...) and to finish the FTP Explorer sample that comes with it.
