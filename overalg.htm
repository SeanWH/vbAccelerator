 6  Algorithms  
 


This section is a general repositry for VB algorithms and techniques for speeding up common operations.

  • Double Linked-Lists and Hashing:
    • Double-Linked List
    • About Hashing and a Powerful Hash Generation Algorithm
    • Double-Linked List with Hashing
  • Collection for access by index. Demonstrates a technique for speeding up insertion/deletion from an array.

  • Build long strings quickly. Appending many elements to a string can be very slow in VB. If you know your about to append many elements, then pre-allocating memory and accessing it through CopyMemory works in a fraction of the time.
