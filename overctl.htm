Source Code
 1  ActiveX Controls  


 NOTE: this page has been superceded by the version at the new site.



 


This section covers custom controls developed from scratch in VB as well as VB code for extending controls.

  • vbAccelerator S-Grid Control - A new grid control with Outlook style grid support. 15 November 1999
  • vbAccelerator RichEdit 2.0 Control - The first in a series of RichEdit releases from vbAccelerator.
    • Coding with the vbAccelerator RichEdit control
    • Using the Zoom function with RichEdit 3.0
    • Using the Outline function with RichEdit 3.0
    • Getting The Text Object Model (TOM) from RichEdit 3.0
  • vbAccelerator ListBar - A precise emulation of the Outlook bar which includes the basis of an all API ListView implementation. 21 March 2000
  • Free-Threaded In-Progress Indicators - alternatives to the Animation control if you want to indicate an operation in progress but don't know when it will end and don't have a suitable AVI.
    • Knight Rider/Netscape Style Bar
    • Animated Cursor
  • Transparent AVI Player Control - demonstrates how to play any type of AVI and shows how to implement transparency.
  • Skin a VB Form - the NeoCaption component can radically customise the appearance of your VB form.
  • Adding Icons to VB Menus
    Three ActiveX objects with increasing levels of complexity and control over VB menus:
    • The IconMenu DLL - the simplest way to get icons into your VB project. This tight DLL supports every one of VB's menu features whilst allowing you to get icons into your menu with only three lines of code!
    •   Icon Menu control Version 2 - Add icons and take more control over menus - add, insert and delete as many new items, including new submenus, as you want at run time. 10 January 1999
      • Adding menu icons to MDI projects with multiple child menus.
      • Using the Icon Menu control to create a Colour Picker.
    •   Popup Menu ActiveX DLL version 2.01 - Ditch those VB menus entirely! A fresh start allows you to create Start Menu/ICQ menus and also to build the ultimate UI with Explorer/Office style CoolMenus. 30 May 1999
      • Using the cNewMenu DLL to Create Start Menu/ICQ Style Pop-up Menus
      • Owner-Draw Pop-up Menus.
      • Using cNewMenu DLL to create SysTray Menus with Icons.
  • vbAccelerator IconEditor Control - A versatile and re-usable control for editing icons.
  •   Owner Drawn Combo and List Box Control Version 2.1 30 May 1999
  • Hierarchy Selector control 15 August 1999
    Now includes VB Thunder's TreeView background image support.
  • Calendar Control from Mike Gainer
  • TextBoxEx Control from Mike Gainer
  • Flat Controls
    • Give Combo Boxes and Text Boxes a Flat, Office 97 Style. 26 February 1999
    • DevStudio style Flat ListView headers. Class also works with the header of the S-Grid control.
  • Creating drop-down tool windows.
    • Word97 toolbar style table picker
    • vbAccelerator Drop-Down Form Control - allows you to convert any form into a drop-down window which can be optionally be dragged-off and floated.
