    Overview
    This page contains the original release of the vbAccelerator ListBar control - now superceded by release 2. Unfortunately, this version cannot be continued because there is a problem with VB which prevents a new version from being compiled.

    This version is still here in case there are projects which use it and require a copy. Future projects or redevelopments of existing projects should use the new release instead:
