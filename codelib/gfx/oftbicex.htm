  Office Toolbar Icon Explorer Utility
  A quick and simple way to extract and save icons from the MS Office CommandBar object

 

Download the Office Toolbar IconEx Utility and Source code (64kb)

  Before you Begin  
  This project requires the vbAccelerator ImageList control. Make sure you have loaded and registered this control before trying the project.  

This simple utility allows you to read all the images contained in MS Office CommandBar objects exposed by MS applications through automation. If you're trying to get a toolbar together in a hurry, then this code can very useful!

Full source code for this sample is provided, which demonstrates how to

  • Save a PictureBox to a real Win32 Icon, regardless of icon size using the cFileIcon class
  • Create common dialogs in code
  • Use a vbAccelerator ImageList with a Microsoft ListView control
  • Read a bitmap on the clipboard directly into an ImageList using the Clipboard Customiser class.
