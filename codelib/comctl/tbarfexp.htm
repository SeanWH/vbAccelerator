 Explorer Style User Interface Cool Menu Sample

 		 
 

Building a Windows Explorer Style Program using vbAccelerator CoolMenu

 		 
 

Download the Explorer UI Demonstration Project (58kb)

  Before you Begin  
  This project requires the vbAccelerator CoolMenu Toolbar and Rebar Control. Make sure you have loaded and registered this before trying the project.  
  CoolMenu support requires version projects require version 2.00.0010 or higher of the ActiveX Popup Menu DLL. Make sure you have loaded and registered this before trying the project.  
  The vbAccelerator CoolMenu Toolbar and Rebar control requires the SSubTmr.DLL component. Make sure you have loaded and registered this before trying the project.  

This project demonstrates how to use the vbAccelerator CoolMenu Toolbar and Rebar control to create an Explorer/Internet Explorer style User Interface. Two rebar bands are used: one to hold the main toolbars and CoolMenu, and a side bar to hold the "Explorer Bar". The sample also demonstrates using the Toolbar Host control to build an title for the Explorer Bar.
