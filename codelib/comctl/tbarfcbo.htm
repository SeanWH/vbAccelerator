 Office 97 Style User Interface CoolMenu Sample

 		 
 

Building an Office 97 Style Program using vbAccelerator CoolMenu

 		 
 

Download the Office 97 UI Demonstration Project (68kb)

  Before you Begin  
  This project requires the vbAccelerator CoolMenu Toolbar and Rebar Control. Make sure you have loaded and registered this before trying the project.  
  This project requires the SSubTmr.DLL component. Make sure you have loaded and registered this before trying the project.  
  CoolMenu support requires version projects require version 2.00.0010 or higher of the ActiveX Popup Menu DLL. Make sure you have loaded and registered this before trying the project.  

 
 


This project demonstrates how to use the vbAccelerator CoolMenu Toolbar and Rebar control to create a serious, Multiple Document Interface Office 97-style user interface. It demonstrates how to:

  • Create toolbar bands containing multiple controls (toolbars and combo boxes) using the toolbar host control.
    The Toolbar Host control, supplied within the vbAccelerator CoolMenu Toolbar and Rebar control makes this easy. Simply drop an instance of the Toolbar Host control on your form, set its height to the height you need to accommodate the controls and then call the Capture method for each control you want to add to the band. That's it! The Toolbar Host then takes care of automatically sizing the contained controls.

  • Create a CoolMenu which works in an MDI form, also using the toolbar host control.
    On SDI forms, a CoolMenu always remains the same. However, in an MDI application, the CoolMenu must modify itself when the MDI child is maximized, so it also has the child form's control box and min/max/restore buttons. The Toolbar Host control has a property, .MDIToolbar which when set True means the toolbar host control automatically provides this functionality. Just connect a CoolMenu Toolbar to the Toolbar Host control with Capture and full MDI CoolMenu support is yours.

  • Flatten the Combo Boxes within the control so they have that Office 97 style, using the cFlatControl class.
