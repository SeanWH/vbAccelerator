Progress Bar Control

Download the Progress Bar Control with source (33kb)

Your license to the code - what you can do

This Progress Bar control is a lightweight (27kb) replacement control for VB's ProgressBar control provided as part of Microsoft Windows Common Controls OCX. It shows how to implement an Progress Bar control entirely in VB utilising COMCTL32.DLL.

A particularly neat bit of this sample demonstrates how to create a colour picker using an OLE Property page (like the one you choose colours with in COMCTL32.OCX or the OLE_COLOR property picker for a user control) and use it from a VB form. The code is very small and it provides a quick and elegant method of choosing back and fore colours.
