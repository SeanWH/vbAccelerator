Using a Toolbar and Rebar with a Pager Control

Download the Toolbar/Pager Demonstration (47kb)

This project requires:
  SSubTmr.DLL component
  vbAccelerator Rebar and Toolbar control
  vbAccelerator ImageList control
Make sure you have loaded and registered these before trying the project.

This project demonstrates how to host a vbAccelerator toolbar control within a Pager control and then to add the pager/toolbar into a rebar band. This allows you to create IE4 style toolbar bands which the user can scroll backwards and forwards to get at buttons which aren't visible when the form is too small.

It also shows how to set up hot images using the vbAccelerator Image List control.
