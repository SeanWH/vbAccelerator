 3  Icon Library  


 NOTE: this page has been superceded by the version at the new site.



 


The Icon Library is here to help creating good looking applications. It includes most of the standard icons (file, edit, internet) you use most, and some more random ones. Even if you don't find exactly what you want, you'll often get a good starting point for building a better icon!

Related Utilities

  • vbAccelerator Icon Editor Control
    This exceptional control allows you to load and edit icons.
  • Office Toolbar Icon Extractor
    A utility for reading the toolbar pictures from Office applications and saving them as .ICO files. Excellent for creating toolbars quickly!
  • Icon Extractor Utility
    A serious utility for extracting entire icons including all resource sizes and colour depths from executables and DLLs.
